# README for `data/`

Created: 2026-01-15  
Updated: 2026-01-15


## Directory
- `data-wrangling-intermediate/`
    - Subfolder contains own README.
- `GIS-exports/`
    - CSVs exported from Arc GIS Pro (products of geoprocessing); not pushed to GitHub.
    - See [Google Doc notes (Maps v02 tab)](https://docs.google.com/document/d/1gshsoIPIgJ5tUT7jBK2LmrB_ZmTPzvl8P9q5xpuQqKA/edit?tab=t.y8517mh4o6i) for more information.
- `LDC-points/`
    - Shapefiles of the LDC points, created from `scripts/02_LDC-to-shapefile.R`.
    - Not pushed to GitHub.
- `raw/`
    - Raw data; most files not pushed to GitHub.
    - Subfolder contains own README.
- `versions-from-R/`
    - Versions of any data cleaned in R and read back into ArcGIS Pro for analysis (such as LTDL treatment info table or LDC points); not pushed to GitHub.
    - See [Google Doc notes (File versions v02 tab)](https://docs.google.com/document/d/1gshsoIPIgJ5tUT7jBK2LmrB_ZmTPzvl8P9q5xpuQqKA/edit?tab=t.s4y0q3mkrb) for more information.
